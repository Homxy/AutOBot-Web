# อ้างอิง API

เอกสารอ้างอิงฟังก์ชันสาธารณะทั้งหมดใน Autobot Library

---

## Arduino API

### robot.init()

เริ่มต้นหุ่นยนต์ด้วยประเภทการขับเคลื่อนและการตั้งค่ามอเตอร์ที่ระบุ

```cpp
#include "ArduinoRobot.h"

robot.init(type, motorPins, motorCount, deviation);
```

#### พารามิเตอร์

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| :--- | :--- | :--- |
| `type` | `DriveType` | ประเภทการขับเคลื่อน: `DRIVE_DIFFERENTIAL`, `DRIVE_OMNI_3W`, `DRIVE_MECANUM` |
| `motorPins` | `int[]` | อาร์เรย์หมายเลขขา GPIO สำหรับแต่ละมอเตอร์ (ดูตารางด้านล่าง) |
| `motorCount` | `int` | จำนวนมอเตอร์ทั้งหมด (เช่น `2`, `3`, `4`) |
| `deviation` | `int` | ค่าแก้ไขการเบี่ยงเบนเส้นตรง ช่วง: `-100` ถึง `100` |

#### อาร์เรย์ขามอเตอร์ตามประเภทการขับเคลื่อน

| ประเภทการขับเคลื่อน | รูปแบบอาร์เรย์ขา |
| :--- | :--- |
| Differential (2W) | `{FL_IN1, FL_IN2, FR_IN1, FR_IN2}` |
| Differential (4W) / Mecanum | `{FL_IN1, FL_IN2, FR_IN1, FR_IN2, RL_IN1, RL_IN2, RR_IN1, RR_IN2}` |
| Omni (3W) | `{M1_IN1, M1_IN2, M2_IN1, M2_IN2, M3_IN1, M3_IN2}` |

#### คู่มือค่าเบี่ยงเบน

| ค่า | ผลลัพธ์ |
| :--- | :--- |
| `-100` ถึง `-1` | ลดความเร็วมอเตอร์ **ขวา** — ใช้เมื่อหุ่นยนต์เบี่ยงไป **ซ้าย** |
| `0` | ไม่มีการแก้ไข |
| `1` ถึง `100` | ลดความเร็วมอเตอร์ **ซ้าย** — ใช้เมื่อหุ่นยนต์เบี่ยงไป **ขวา** |

### robot.drive()

สั่งให้หุ่นยนต์เคลื่อนที่ในทิศทางที่กำหนดด้วยความเร็วที่กำหนด พร้อมการหมุนเพิ่มเติม

```cpp
robot.drive(speed, direction, angular);
```

#### พารามิเตอร์

| พารามิเตอร์ | ชนิด | ช่วง | คำอธิบาย |
| :--- | :--- | :--- | :--- |
| `speed` | `int` | `0` – `100` | ความเร็วเคลื่อนที่ (0 = หยุด) |
| `direction` | `int` | `0` – `360` | ทิศทางเป็นองศา `0` = ไปข้างหน้า, `180` = ถอยหลัง |
| `angular` | `int` | `-100` – `100` | ความเร็วหมุน `-100` = หมุนซ้ายเต็ม, `100` = หมุนขวาเต็ม |

### การตั้งค่าการสื่อสาร Pi

เริ่มต้นการเชื่อมต่อ UART ระหว่าง ESP32 และ Raspberry Pi

```cpp
// ใน setup()
PiSerial.begin(baudRate, SERIAL_8N1, PI_RX_PIN, PI_TX_PIN);
piComm.init();
```

| พารามิเตอร์ | ชนิด | ค่าเริ่มต้น | คำอธิบาย |
| :--- | :--- | :--- | :--- |
| `baudRate` | `int` | `115200` | อัตรา Baud ของ Serial (ต้องตรงกับการตั้งค่าฝั่ง Pi) |
| `PI_RX_PIN` | `int` | `16` | ขา GPIO ของ ESP32 ที่เชื่อมต่อกับ TX ของ Pi |
| `PI_TX_PIN` | `int` | `17` | ขา GPIO ของ ESP32 ที่เชื่อมต่อกับ RX ของ Pi |

```cpp
// ใน loop() — จำเป็นสำหรับประมวลผลคำสั่งขับเคลื่อนที่เข้ามา
piComm.handle();
```

### piComm.requestHuman()

ร้องขอให้ Raspberry Pi เริ่มโหมด **Human Tracking**

```cpp
piComm.requestHuman(baseSpeed, turnGain, boxSize, deadzone);
```

| พารามิเตอร์ | ชนิด | ค่าเริ่มต้น | คำอธิบาย |
| :--- | :--- | :--- | :--- |
| `baseSpeed` | `int` | `0` | ความเร็วเดินหน้าขณะติดตาม `0` = หมุนอย่างเดียว, `100` = เคลื่อนที่ไปข้างหน้าเสมอ |
| `turnGain` | `float` | `0.5` | ความไวในการหมุน ค่ายิ่งสูง ตอบสนองยิ่งเร็ว (เสี่ยง Overshoot) |
| `boxSize` | `int` | `30000` | ขนาดพื้นที่ Bounding Box เป้าหมาย ค่ายิ่งใหญ่ หุ่นยนต์เข้าใกล้ยิ่งขึ้น |
| `deadzone` | `int` | `5000` | Deadzone ของมุม — หุ่นยนต์จะไม่หมุนถ้าเป้าหมายอยู่ในเกณฑ์นี้ |

---

### piComm.requestLine()

ร้องขอให้ Raspberry Pi เริ่มโหมด **Line Tracking**

```cpp
piComm.requestLine(baseSpeed, turnGain, thresh, color, defS, defD, defA);
```

| พารามิเตอร์ | ชนิด | ค่าเริ่มต้น | คำอธิบาย |
| :--- | :--- | :--- | :--- |
| `baseSpeed` | `int` | `0` | ความเร็วเดินหน้าขณะติดตาม `0` = หมุนอย่างเดียว |
| `turnGain` | `float` | `0.5` | ความไวในการหมุน |
| `thresh` | `int` | `100` | ค่า Threshold สำหรับ Image Mask ที่ใช้ในการตรวจจับเส้น |
| `color` | `String` | `"black"` | สีของเส้นที่จะติดตาม |
| `defS` | `int` | `0` | ความเร็วสำรองเมื่อไม่พบเส้น |
| `defD` | `int` | `0` | ทิศทางสำรอง (องศา) เมื่อไม่พบเส้น |
| `defA` | `int` | `0` | ความเร็วเชิงมุมสำรองเมื่อไม่พบเส้น |

---

### piComm.setDriveType()

เปลี่ยนประเภทการขับเคลื่อนขณะทำงาน (เช่น สลับหุ่นยนต์ Mecanum เป็นการเคลื่อนที่ด้านข้าง)

```cpp
piComm.setDriveType(type);  // "diff", "omni", or "mac"
```

---

### piComm.requestStop()

หยุดการประมวลผล AI ทั้งหมดบน Raspberry Pi

```cpp
piComm.requestStop();
```

## Raspberry Pi Serial Protocol

ESP32 สื่อสารกับ Raspberry Pi โดยใช้โปรโตคอล Serial แบบข้อความ

### รูปแบบข้อความ

```
:M|parameter=value|parameter=value|...
```

**ตัวอย่าง:**

```
:M|spd=50|sen=0.8|LN=1
```

คำสั่งนี้ตั้งค่าความเร็วเป็น `50`, ความไวเป็น `0.8` และเริ่มโหมด **Line Tracking**

### ตารางอ้างอิงพารามิเตอร์

| พารามิเตอร์ | ชนิด | คำอธิบาย |
| :--- | :--- | :--- |
| `LN` | `int` | โหมด Line Tracking: `0` = หยุด, `1` = เริ่ม |
| `HM` | `int` | โหมด Human Tracking: `0` = หยุด, `1` = เริ่ม |
| `spd` | `int` | ความเร็วเคลื่อนที่ (`0`–`100`) ใช้ได้ทั้งสองโหมด |
| `sen` | `float` | ความไวในการหมุน ใช้ได้ทั้งสองโหมด |
| `thr` | `int` | Image Threshold (`0`–`100`) สำหรับ Line Tracking เท่านั้น |
| `type` | `String` | ประเภทการขับเคลื่อน: `"diff"`, `"omni"`, หรือ `"mac"` |
| `box` | `int` | ขนาด Bounding Box เป้าหมาย สำหรับ Human Tracking เท่านั้น |
| `dzone` | `int` | Angular Deadzone ใช้ได้ทั้งสองโหมด |
| `col` | `String` | สีเส้น (เช่น `"black"`) สำหรับ Line Tracking เท่านั้น |
| `def` | `String` | ค่าสำรองเป็น `speed,direction,angular` (เช่น `"50,180,0"`) สำหรับ Line Tracking เท่านั้น |
