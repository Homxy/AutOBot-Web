# ตัวอย่าง

หน้านี้มี Sketch ที่พร้อมใช้งานสำหรับการตั้งค่า Autobot ทั่วไป แต่ละตัวอย่างสาธิตประเภทการขับเคลื่อนและชุดฟีเจอร์ที่แตกต่างกัน

## การติดตั้ง

1. ดาวน์โหลด [`AutOBot Library`](https://github.com/Homxy/AutOBot)
2. วางไฟล์ในโฟลเดอร์โปรเจกต์ Arduino ของคุณ
3. เพิ่มไว้ที่ด้านบนของ Sketch:

```cpp
#include "AutOBot.h" // หากต้องการใช้ไลบรารีโดยไม่มี AI
#include "AutOBotAI.h" // หากต้องการใช้ไลบรารีที่มี AI
```

## Differential Drive (2 ล้อ)

### Differential พร้อม Raspberry Pi

```cpp
#include <AutOBot.h>
#include <AutOBotAI.h>

AutOBot myBot;
AutOBotAI myAI;

void setup() {
  Serial.begin(115200);
  Serial2.begin(115200, SERIAL_8N1, myAI.rx, myAI.tx); // RX, TX สำหรับติดต่อกับ AI
  
  myBot.begin(DRIVE_DIFFERENTIAL, 0);
  myAI.begin(Serial, myBot);

}

void loop() {
  myAI.handle();
}
```

### Differential ไม่มี Raspberry Pi

Sketch ขั้นต่ำที่ขับเคลื่อนหุ่นยนต์ไปข้างหน้าโดยไม่ใช้ฟีเจอร์ AI

```cpp
#include <AutOBot.h>

AutOBot myBot;

void setup() {  
  myBot.begin(DRIVE_DIFFERENTIAL, 0);
}

void loop() {
  myBot.drive(0, 100, 0);
}
```

## Mecanum Drive (4 ล้อ)

### Mecanum พร้อม Raspberry Pi

```cpp
#include <AutOBot.h>
#include <AutOBotAI.h>

AutOBot myBot;
AutOBotAI myAI;

void setup() {
  Serial.begin(115200);
  Serial2.begin(115200, SERIAL_8N1, myAI.rx, myAI.tx); // RX, TX สำหรับติดต่อกับ AI
  
  myBot.begin(DRIVE_MECANUM, 0);
  myAI.begin(Serial, myBot);
  myAI.requestHuman(50, 0.5, 30000, 5000);

}

void loop() {
  myAI.handle();
}
```

### ลำดับ AI แบบจับเวลา

ตัวอย่างนี้สาธิตวิธีการเรียงลำดับโหมด AI หลายโหมดโดยใช้ Timer:

1. **เฟส 1** — ติดตามมนุษย์เป็นเวลา **10 วินาที**
2. **เฟส 2** — ติดตามเส้นเป็นเวลา **15 วินาที**
3. **เฟส 3** — หยุด

```cpp
#include <AutOBot.h>
#include <AutOBotAI.h>

AutOBot myBot;
AutOBotAI myAI;

unsigned long startTime;
bool phaseLineStarted = false;
bool phaseStopStarted = false;

void setup() {

  Serial.begin(115200);
  Serial.println("ESP32 Ready (AutOBot Mode)");

  Serial2.begin(115200, SERIAL_8N1, myAI.rx, myAI.tx);

  myBot.begin(DRIVE_MECANUM, 0);

  // Bind AI module with Serial2 and robot
  myAI.begin(Serial2, myBot);

  // ── เฟส 1: ติดตามมนุษย์ (10s) ────────────
  Serial.println("PHASE 1: Human Tracking (10s)");
  myAI.requestHuman(30, 0.6, 30000, 5000);

  startTime = millis();
}

void loop() {

  // ประมวลผลข้อมูล AI ขาเข้าเสมอ
  myAI.handle();

  unsigned long elapsed = millis() - startTime;

  // ── หลังจาก 10 วินาที → ติดตามเส้น ───────────────
  if (!phaseLineStarted && elapsed > 10000) {
    Serial.println("PHASE 2: Line Tracking (15s)");

    myAI.requestLine(30, 0.8, 100, "black", 0, 0, 0);

    phaseLineStarted = true;
  }

  // ── หลังจาก 25 วินาที → หยุด ────────────────────────
  if (!phaseStopStarted && elapsed > 25000) {
    Serial.println("PHASE 3: Sequence complete. Stopping.");

    myAI.requestStop();

    phaseStopStarted = true;
  }
}
```

### Mecanum ไม่มี Raspberry Pi

Sketch Mecanum ขั้นต่ำที่ขับเคลื่อนหุ่นยนต์ไปข้างหน้าโดยไม่ใช้ AI

```cpp
#include <AutOBot.h>

AutOBot myBot;

void setup() {
  myBot.begin(DRIVE_MECANUM, 0);
}

void loop() {
  myBot.drive(0, 100, 0);
}
```

## คู่มือด่วนโหมด AI

คุณสามารถเรียกฟังก์ชันเหล่านี้ได้ทุกที่ใน `loop()` — ภายใน `if`, `else`, `while`, Timer หรือบล็อกเงื่อนไขอื่นๆ

### การติดตามเส้น

```cpp
myAI.requestLine(30, 0.8, 100, "black");
```

หุ่นยนต์จะเริ่มติดตามเส้น ดูรายละเอียดพารามิเตอร์ทั้งหมดที่ [อ้างอิง API](/docs/ai/reference_th)

### การติดตามมนุษย์

```cpp
myAI.requestHuman(30, 0.6, 30000, 5000);
```

หุ่นยนต์จะเริ่มติดตามมนุษย์ที่ตรวจพบ ดูรายละเอียดพารามิเตอร์ทั้งหมดที่ [อ้างอิง API](/docs/ai/reference_th)
