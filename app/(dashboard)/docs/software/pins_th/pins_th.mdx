# บล็อกควบคุมพิน

## ตั้งค่าสถานะพิน (`pin_set`)

กำหนดให้พินดิจิทัลเปิด (HIGH) หรือปิด (LOW)

### Arduino Generates

```cpp
digitalWrite(pin, state);
```

---

## ตั้งค่าโหมดพิน (`pin_out`)

กำหนดให้พินเป็น OUTPUT หรือ INPUT

### Arduino Generates

```cpp
pinMode(pin, mode);
```

---

## หน่วงเวลา (`wait_ms`)

หยุดพักการทำงานตามจำนวนมิลลิวินาทีที่กำหนด

### Arduino Generates

```cpp
delay(ms);
```
