# Logic Blocks

## If / Else Condition (`controls_if`)

Used to execute different code depending on a condition.

### Arduino Generates

```cpp
if (condition) {
  // Code
} else {
  // Code
}
```

---

## Comparison (`logic_compare`)

Compares two values (e.g., `==`, `!=`, `<`, `>`).

### Arduino Generates

```cpp
value1 == value2
```

---

## Boolean (`logic_boolean`)

Represents a True or False value.

### Arduino Generates

```cpp
true / false
```
