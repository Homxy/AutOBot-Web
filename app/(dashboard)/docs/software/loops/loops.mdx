# Loop Blocks

## Repeat X Times (`controls_repeat_ext`)

Repeats the code inside a specific number of times.

### Arduino Generates

```cpp
for (int i = 0; i < times; i++) {
  // Code
}
```

---

## While / Until (`controls_whileUntil`)

Loops code while or until a condition is met.

### Arduino Generates

```cpp
while (condition) {
  // Code
}
```
