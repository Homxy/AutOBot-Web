# บล็อกตรรกะ

## เงื่อนไข If / Else (`controls_if`)

ใช้สำหรับรันโค้ดที่แตกต่างกันขึ้นอยู่กับเงื่อนไข

### Arduino Generates

```cpp
if (condition) {
  // Code
} else {
  // Code
}
```

---

## การเปรียบเทียบ (`logic_compare`)

เปรียบเทียบค่าสองค่า (เช่น `==`, `!=`, `<`, `>`)

### Arduino Generates

```cpp
value1 == value2
```

---

## ตรรกะ (`logic_boolean`)

ค่าความจริง (True) หรือเท็จ (False)

### Arduino Generates

```cpp
true / false
```
