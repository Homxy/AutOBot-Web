# บล็อกลูป (การวนซ้ำ)

## ทำซ้ำ X ครั้ง (`controls_repeat_ext`)

ทำซ้ำโค้ดตามจำนวนครั้งที่กำหนด

### Arduino Generates

```cpp
for (int i = 0; i < times; i++) {
  // Code
}
```

---

## ทำซ้ำตราบใดที่ / จนกระทั่ง (`controls_whileUntil`)

วนลูปโค้ดตราบใดที่เงื่อนไขเป็นจริง (หรือจนกว่าจะเป็นจริง)

### Arduino Generates

```cpp
while (condition) {
  // Code
}
```
