# Pin Control Blocks

## Set Pin State (`pin_set`)

Sets a specific digital pin to ON (HIGH) or OFF (LOW).

### Arduino Generates

```cpp
digitalWrite(pin, state);
```

---

## Set Pin Mode (`pin_out`)

Sets a pin as an OUTPUT or INPUT.

### Arduino Generates

```cpp
pinMode(pin, mode);
```

---

## Wait / Delay (`wait_ms`)

Pauses execution for a specified number of milliseconds.

### Arduino Generates

```cpp
delay(ms);
```
